Descripción del Proyecto

Zettium es una aplicación web diseñada para empresas de servicios técnicos (aires acondicionados, calefacción, electricidad, plomería, gas y energía solar).

Facilita la gestión centralizada de clientes, presupuestos, obras y visitas técnicas, con trazabilidad, automatización de cálculos y generación de presupuestos en PDF.

Funcionalidades Principales

  • ABM de Clientes y Presupuestos
  • Gestión de Ítems reutilizables
  • Control de Estados para obras
  • Agenda y recordatorios para visitas
  • Exportación a PDF

Arquitectura

Desarrollo siguiendo principios de Arquitectura Clean, separación por capas y uso de Entity Framework Core para persistencia.
