Zettium

Sistema Integral de Gestión Técnica para presupuestos, obras y visitas técnicas.

Proyecto Fullstack · .NET · Blazor · SQL Server

Descripción del Proyecto

Zettium es una aplicación web diseñada para empresas de servicios técnicos especializados en aires acondicionados, calefacción, electricidad, plomería, gas y energía solar.

El sistema facilita la gestión centralizada de clientes, presupuestos, obras y visitas técnicas, ofreciendo trazabilidad completa, automatización de cálculos y una interfaz moderna enfocada en la productividad operativa.

Funcionalidades Principales

  • ABM Completo: Gestión integral de clientes, presupuestos y obras.
  • Gestión de Ítems: Reutilización de materiales y servicios precargados.
  • Control de Estados: Seguimiento de obras (iniciada, en proceso, finalizada).
  • Agenda Visual: Programación y recordatorios de visitas técnicas.
  • Historial Técnico: Registro detallado por cliente.
  • Generación de PDFs: Presupuestos profesionales automatizados.
  • Filtros y Búsqueda: Dinámicos y en tiempo real.
  • Gestión Financiera: Control de costos, totales y materiales.
  • Autenticación: Preparado para Identity Framework.

Arquitectura

El proyecto se desarrolló siguiendo los principios de Arquitectura Clean, con separación clara de responsabilidades para garantizar escalabilidad y mantenibilidad.

Zetta.BD

Modelo de datos, entidades y configuración de Entity Framework Core.

Zetta.Shared

DTOs, enums y contratos compartidos entre frontend y backend.

Zetta.Server

API REST en .NET con servicios, repositorios y AutoMapper.

Zetta.Client

Frontend Blazor WebAssembly con diseño responsive.

Tecnologías Utilizadas

.NET 8 / 9 Blazor WebAssembly Entity Framework Core SQL Server AutoMapper Bootstrap HTML · CSS Arquitectura Clean
